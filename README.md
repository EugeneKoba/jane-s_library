## Jane's Library

Full Website Project for Jane's Library Instagram page, containing the following:

- Home Page
- About Section
- Blog Page
- Book Storage Page
- Contact Us Page
- Search Books Page.

Made with HTML, Bootstrap (CSS) & Vanilla Javascript. Website Icons Created with Adobe Photoshop & Illustrator.
